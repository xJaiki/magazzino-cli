# Magazzino v1.2.1 ‚Äî 2026-06-10

Doctor catch-up release.

## Changed
- **`mag doctor` knows every optional dependency** introduced in 1.2.0: it now
  checks `gh`, `tmux`, `rg`, `bat`, `glow` and `xdg-open`, each with a note on
  exactly which feature it enables (e.g. tmux ‚Üí `mag t` / `^t`). Missing
  optional tools never fail the health check ‚Äî they only disable their
  feature, and the report says so.
- Doctor also reports the **status-dot cache**: its age in seconds when
  present, or a hint to run `mag refresh` when missing.
