# Magazzino v1.1.0 — 2026-06-10

Visual refresh: the dashboard and every picker get a cleaner, more readable
layout. No behavior changes, no new dependencies.

## Changed
- **Reverse layout everywhere**: the list starts from the top with the prompt
  above it — natural reading order (was bottom-up).
- **Framed panel**: rounded border with a title label showing version and
  project count; labeled preview pane with its own left border.
- **Column-aligned list**: pin marker, `[root]` (multi-root), category and
  name are padded into clean columns; project **tags render inline** in dim
  (`#uni #frontend`) so labels are visible without opening the preview.
- **Full-line selection highlight** (`--highlight-line`), scrollbar, list
  separator and inline match counter.
- **Two-line key legend** in the dashboard header, grouped by intent
  (navigate vs act) with highlighted keys — replaces the single cramped line.
- **Repo state in the preview header**: current branch, `↑ahead ↓behind` and
  a dirty/clean badge (`● 3 modified` / `● clean`) for the selected project.
- Consistent styling across all pickers (jump, category, dirty-repo jump,
  GitHub clone) via a shared fzf wrapper.

## Compatibility
The styled layout needs fzf ≥ 0.53; on older versions Magazzino falls back to
a plain picker automatically.
