# Magazzino v1.0.0 — 2026-06-10

The definitive release. This version supersedes both the legacy `mag` series
(up to 1.2.0) and the 2.0 preview: same tool, one codebase, fresh versioning.

## Core
- **Dashboard**: bare `mag` opens a full-screen interactive view of every
  project. Keys: `enter` jump, `ctrl-e` editor, `ctrl-w` web, `ctrl-y` pin,
  `ctrl-a` archive, `ctrl-s` status, `ctrl-p` pull, `esc` quit.
- **Navigation**: fuzzy picker with preview (description, tags, git log,
  files), recency ordering, direct jump on unique match (`mag j name`),
  tag filtering (`mag j @tag`), ping-pong jump-back (`mag -`).
- **Lifecycle**: `new` (named templates or URL), `clone` (history preserved,
  `user/repo` shorthand, gh-powered interactive pick), `mv` (projects and
  whole categories), `archive`/restore, guarded `rm`.
- **Metadata**: `pin` (favorites on top), `tag`, `desc` — flat TSV store,
  shown in previews.
- **Sync**: `status` with dirty counts and unpushed/behind markers (`s j`
  jump into a dirty repo, `s f` fetch first), bulk fast-forward `pull`.
- **Multiple roots**: `mag config root add/rm/list` merges several project
  directories into every picker and scan.
- **Tool**: configurable editor, named templates, self-update from the source
  repo, in-terminal changelogs, doctor (deps, roots, strays, plugins),
  bash/zsh completion, plugin system (`mag_cmd_<name>()` files in
  `~/.config/magazzino/commands/`).

## Fixed since the 2.0 preview
- `mag web` / `ctrl-w` now falls back to the first configured remote when
  `origin` is missing, and the error message states clearly that the project
  has no remote at all.
- Dashboard actions that fail (e.g. web on a remote-less project) pause so
  the message is readable before the list redraws.
- Dashboard key parsing hardened against custom `FZF_DEFAULT_OPTS`.

## Migration
The installer retires legacy v1 and 2.0-preview source lines from your shell
rc, removes their stale app files, migrates recency/metadata from the preview,
and reuses the existing config in `~/.config/magazzino/` unchanged.
